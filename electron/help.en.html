BitResume Documentation

Quick guide to CV editing, data storage, and PDF export (desktop app).

What is BitResume?

BitResume is a CV builder that supports two language versions (PL and EN) in a single template file. You can switch visual modes, edit content inline, and export to PDF.

Language and CV content

  • PL / EN — the toolbar switch changes the currently edited language version; both are stored in one JSON file.
  • Edit mode — enables inline editing (text fields, lists, add/remove/reorder actions). While it is on, the toolbar button is clearly highlighted; the accent color depends on the active theme (dark / mixed / print). With edit mode off, the CV looks like a read-only preview.

Visual modes

  • Mixed mode (default) — bright CV card with navy accents.
  • Print mode — print-friendly bright layout.
  • Dark mode (web) — dark shell and dark CV card.

Save and load data

  • Save — saves your current template (PL+EN) to the configured location.
  • Save as — sets output file name (for example template.json) and target folder (desktop: typically Documents / CV_Data).
  • Load template — opens a JSON file from disk and replaces current data (if format is valid).
  • On first run, the app can create a default template.json in the CV data folder; you can overwrite it anytime.

Print to PDF

Print to PDF exports the currently visible CV and respects active color mode. Editing can be temporarily disabled while export is in progress.

Profile photo

In edit mode you can upload, crop, and remove profile photo.

Sections and order

Main blocks (experience, projects, certifications, etc.) can be managed in edit mode. Lists and skills can also be added, removed, and reordered.

Help menu
